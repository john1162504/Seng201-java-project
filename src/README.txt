How to import this source code to eclipse
1.Extract this zip file
2.Create a new empty Java project into the workspace
3.Right click on the source folder and choose import...->General->File system
4.choose your files
5.done!

How to run this JAR program
1.Ensure Java 17 is installed on your machine
2.Open Extract this zip file
3.Run "java cch235_rli107SumoAgent.jar" in your terminal
